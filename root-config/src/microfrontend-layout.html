Welcome to a Micro-frontend App

This is a demo of a micro-frontend app built with Single-Spa. In total three react application was merged together to create this application. They are Rick and Morty app, Game of Thrones app and a Navbar. Click on the navbar to see them.

Rick and Morty and Game of Thrones apps were initially created using create-react-app then it was converted to a single-spa app. The steps to convert a react app to a single-spa app will be explained in a blog post.

You can see the source code of each application in this Github repository.
